# APK Container — Безопасная песочница для Android

**APK Container** — это приложение-песочница, которое позволяет безопасно анализировать и запускать APK-файлы в изолированной среде на Android. Создано для защиты пользователей от мошеннических приложений и вирусов.

## Возможности

- **Анализ APK до установки** — показывает все запрашиваемые разрешения с оценкой риска
- **Цветовая маркировка риска** — красный/оранжевый/жёлтый/зелёный для каждого разрешения
- **Обнаружение подозрительных комбинаций** — предупреждения вроде «читает SMS + доступ в интернет»
- **Песочница через Work Profile** — изоляция приложений через встроенный механизм Android
- **Мониторинг сетевой активности** — VPN-мониторинг показывает куда приложение отправляет данные
- **Светлая и тёмная тема** — с поддержкой Dynamic Color (Material You)
- **Русский и английский язык**

## Технологии

- **Kotlin** + **Jetpack Compose** + **Material 3**
- **MVVM** + Clean Architecture
- **Hilt** (DI), **Room** (БД), **Compose Navigation**
- **VpnService** для мониторинга сети
- **DevicePolicyManager** для Work Profile

## Требования

- Android 8.0 (API 26) и выше
- ~95% всех Android-устройств

## Сборка

```bash
# Клонировать репозиторий
git clone https://github.com/<your-username>/Apk-container.git
cd Apk-container

# Собрать debug-версию
./gradlew assembleDebug

# APK будет в app/build/outputs/apk/debug/
```

Для сборки нужен Android SDK с platform 35 и build-tools 35.0.0.

## Структура проекта

```
app/src/main/java/com/apkcontainer/
├── data/          — база данных, репозитории, анализатор APK
├── domain/        — доменные модели и интерфейсы
├── sandbox/       — управление Work Profile
├── service/       — VPN-сервис для мониторинга сети
├── ui/            — экраны, тема, навигация, компоненты
└── util/          — утилиты, описания разрешений
```
